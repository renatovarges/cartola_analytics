
TABELA DE GOLEIROS — DOCUMENTO DESCRITIVO FINAL (TCC)

1. OBJETIVO DA TABELA
Esta tabela tem como objetivo analisar o desempenho de GOLEIROS no Cartola a partir
de dados objetivos, organizados por confronto (CASA x FORA), considerando:

- Scouts CONQUISTADOS pelo time do goleiro
- Scouts CEDIDOS pelo time adversário
- Influência do mando de campo
- Comparação direta entre os dois lados do confronto

--------------------------------------------------
3. PRINCÍPIOS GERAIS (IMPORTANTÍSSIMO)

3.1. Soma x Média
- SOMA: regra padrão para quase todos os scouts
- MÉDIA: usada APENAS em dois casos:
  a) CHUTES PARA MARCAR UM GOL (CHUT.PM)
  b) % DE DEFESAS (%DE)

3.2. SG (Saldo de Gols / Clean Sheet)
- SG NÃO é scout individual
- SG é um bônus DO TIME
- Em cada jogo:
  • Se o time NÃO sofreu gol → SG = 1
  • Se sofreu gol → SG = 0
- Portanto:
  - SG conquistado = soma de jogos sem sofrer gols
  - SG cedido = soma de jogos em que o adversário não sofreu gols
- NÃO existe soma de SG por jogador

--------------------------------------------------
4. ESTRUTURA GERAL DA TABELA

A TABELA DE GOLEIROS TEM UMA ESTRUTURA UM POUCO DIFERENTE, POIS TEM UMA DIVISÃO DAS COLUNAS NA PARTE SUPERIOR QUE DIVIDE OS SCOUTS EM AMEAÇAS X OPORTUINIDADES EM AMBOS OS LADOS DA TABELA.(EXEMPLO VISUAL EM ANEXO).


A leitura é SIMÉTRICA:
MANDANTE (lado esquerdo) x VISITANTE (lado direito)

--------------------------------------------------

MONTAGEM DA TABELA:

LADO ESQUERDO

SESSÃO "AMEAÇAS"

SCOUTS UTILIZADOS (GOLEIROS)

1. CHUT.AG — Chutes a Gol - Quantidade de finalizações no alvo.

- CONQUISTADO PELO VISITANTE(COF):
  Soma dos chutes no alvo que o time adversário do goleiro conquistou.
- CEDIDO EM CASA PELO MANDANTE(CDC):
 Soma dos chutes no alvo sofridos pelo time do goleiro.

2. CHUT.PM — Chutes para Marcar um Gol
Relação entre finalizações e gols sofridos, ou seja, é a média de finalizações totais que um time soma para cada gol que marca.

- CONQUISTADO (COF):
  Média de chutes necessários para o adversário do goleiro (visitante) marcar um gol.
- CEDIDO (CDC):
  Média de chutes necessários para os adversários marcarem um gol no time mandante.

3. GOLS
Quantidade de gols sofridos.

- CONQUISTADO (COF):  Soma de gols MARCADOS PELO TIME VISITANTE.
- CEDIDO (CDC): Soma de gols CEDIDO PELOS MANDANTE AOS SEUS ADVERSÁRIOS.

--------------------------------------------------
SESSÃO "OPORTUNIDADES"

4. DE — Defesas
Quantidade total de defesas realizadas.

- CONQUISTADO (COC):  Soma de defesas conquistadas pelo time do goleiro.
- CEDIDO (CDF): Soma de defesas que o visitante cedeu aos goleiros adversários.

5. SG — Clean Sheet - Bônus por jogo sem sofrer gols.

- CONQUISTADO (COC): Soma de jogos em que o time mandante NÃO sofreu gols.
- CEDIDO (CDF): Soma de jogos em que o visitante NÃO marcou gols.

6. %DE — Percentual de Defesas. Eficiência do goleiro. É uma MÉDIA.

- CONQUISTADO (COC): Média do percentual de defesas do goleiro do time mandante para que o visitante marque um gol. Média de defesas por gol sofrido.
- CEDIDO (CDF): Média do percentual de defesas dos goleiros adversários do visitante para sofrer um gol. 

--------------------------------------------------
MANDO DE CAMPO — CASA x FORA

Cada linha representa UM confronto.
A coluna central da tabela contém:
- Escudo do time mandante
- Escudo do time visitante

Todos os scouts são recalculados considerando:
- Desempenho do time quando atua EM CASA
- Desempenho do time quando atua FORA

--------------------------------------------------
LADO ESQUERDO

SESSÃO "AMEAÇAS"


1. CHUT.AG — Chutes a Gol - Quantidade de finalizações no alvo.

- CONQUISTADO PELO MANDANTE(COC): Soma dos chutes no alvo que o time mandante.

- CEDIDO EM CASA PELO VISITANTE(CDF): Soma dos chutes no alvo sofridos pelo time do visitante.


2. CHUT.PM — Chutes para Marcar um Gol
Relação entre finalizações e gols sofridos, ou seja, é a média de finalizações totais que um time soma para cada gol que marca.

- CONQUISTADO (COC): Média de chutes necessários para o mandante marcar um gol.
- CEDIDO (CDF): Média de chutes necessários para os adversários do visitante marcarem um gol.

3. GOLS - Quantidade de gols sofridos.

- CONQUISTADO (COC): Soma de gols MARCADOS PELO TIME MANDANTE.
- CEDIDO (CDF): Soma de gols CEDIDO PELOS VISITANTE AOS SEUS ADVERSÁRIOS.

--------------------------------------------------
SESSÃO "OPORTUNIDADES"

4. DE — Defesas
Quantidade total de defesas realizadas.

- CONQUISTADO (COF):  Soma de defesas conquistadas pelo goleiro do time visitante.
- CEDIDO (CDC): Soma de defesas que o mandante cedeu aos goleiros adversários.

5. SG — Clean Sheet - Bônus por jogo sem sofrer gols.

- CONQUISTADO (COF): Soma de jogos em que o time VISITANTE NÃO sofreu gols.
- CEDIDO (CDC): Soma de jogos em que o MANDANTE NÃO marcou gols.

6. %DE — Percentual de Defesas. Eficiência do goleiro. É uma MÉDIA.

- CONQUISTADO (COF): Média do percentual de defesas do goleiro do time visitante por gol sofrido. Média de defesas por gol sofrido.
- CEDIDO (CDC): Média percentual de defesas permitidas pelo time mandante para o goleiro adversário por gol sofrido. Média de defesas cedida pelo mandante para os goleiros adversários sofrerem um gol.

8. FINALIDADE ANALÍTICA

A tabela permite identificar:
- Goleiros com alto volume de defesas possíveis
- Confrontos com maior chance de SG
- Jogos com tendência a muitos chutes ou poucos gols
- Melhores relações risco x retorno
